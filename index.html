Petit rappel

Pour les verbes réguliers en -ER, le participe passé se forme avec : radical + é Exemples : regarder → regardé · visiter → visité · travailler → travaillé.

Lisez la phrase, puis glissez le bon participe passé dans chaque espace vide.
